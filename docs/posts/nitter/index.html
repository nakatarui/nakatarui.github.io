「Nitterはまだ使えるの?」 2024/12月 Twitter をログインなしで閲覧する。

Mon, Dec 9, 2024

こんにちは。
Nitter はまだ使えるのか? そもそも Nitter とは何なのか?
そんな疑問を記事にしました。

Nitterとは?

Nitter は以下のような特徴を備えた オープンソースかつフリーなTwitterビューアーです。

  • javascriptを使わなくても動作し、広告がない
  • ユーザーはTwitterに直接触れることなく使用できる
  • Twitterのトラッキングを防ぐことができる
  • Twitterの非公式APIを使っている(レート制限がなく、デベロッパーアカウントも必要ない)
  • 軽量であること
  • RSSフィードに対応している
  • 公式のWebインスタンス以外にも、非公式のインスタンスがある

Twitterの仕様変更などの理由で公式のNitterプロジェクトは2024/2月に開発を終了しました。
しかし今でもアクティブなインスタンスがいくつか残っています。

Nitterは今でも使えるのか

現在、Nitter インスタンスのほとんどが動かない状況です。
それでも、安定的に動いてくれるインスタンスが nitter.poast.org です。
非常にレスポンスが良く、ほとんどエラーを吐くこともありません。一つ注意点としては、ブラウザの認証のために javascript と クッキー が必要となることです。

他にも動くインスタンスはありますが、不安定でまともに使えないものばかりなので今は nitter.poast.org を使うのが一番です。
ちなみに、 Nitter インスタンスのサーバーの動作状況を確認できる status.d420.deというサイトがあります。 これを見ると上述の通り、ほとんどのサーバーが不安定、もしくは動作不能な状態になっていることがわかります。
